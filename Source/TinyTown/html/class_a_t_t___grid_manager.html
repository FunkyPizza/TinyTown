void ATT_GridManager::SpawnTiles ( int  x,
int  y,
FVector  center,
float  distance 
)
protected

Spawns a grid of tile instances separated by distance in a x by y grid, centered around Center.

Calculates all tiles locations & stores them in tileLocations, then spawns the instances using tileSpriteNormal. The tiles will be spawned around a center point.

Parameters
xSize X in tiles of the grid.
ySize Y in tiles of the grid.
CenterCenter point world location.
Distancebetween each tiles in UE units.
